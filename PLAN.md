# EasyLayout - 공간 배치도 편집기 개발 계획

## 프로젝트 개요

드래그 앤 드랍 기반의 공간 배치도 편집 도구.
사용자가 도형을 배치하고, 크기/색상/텍스트를 설정하여 직관적으로 공간 레이아웃을 구성할 수 있다.

---

## 기술 스택

### Phase 1 (현재)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **번들러**: Webpack 5
- **렌더링 방식**: Canvas API 또는 SVG (검토 후 결정)

### Phase 2 (추후)
- **Backend**: 미정 (Node.js/Express 또는 다른 서버)
- **Database**: 미정
- **API**: REST 또는 GraphQL

---

## Phase 1 기능 명세

### 1. 캔버스 (편집 영역)
- 격자(grid) 배경 표시
- 줌 인/아웃 지원 (마우스 휠)
- 패닝(pan) 지원 (빈 영역 드래그)
- 캔버스 크기 설정 (실제 공간 크기 입력, 예: 10m x 8m)

### 2. 도형 종류
- 사각형 (Rectangle)
- 원 (Circle)
- 삼각형 (Triangle)
- 선 (Line)
- 커스텀 다각형 (Polygon) — 추후 추가

### 3. 도형 속성
- **크기**: width, height 직접 입력 또는 핸들 드래그로 조절
- **위치**: x, y 좌표 직접 입력 또는 드래그로 이동
- **색상**: fill 색상 선택 (color picker)
- **테두리**: stroke 색상, 두께 설정
- **텍스트**: 도형 내부에 라벨 텍스트 입력, 폰트 크기 설정
- **회전**: 회전 핸들로 자유 회전
- **투명도**: opacity 설정

### 4. 편집 기능
- 도형 선택 (클릭)
- 다중 선택 (Shift+클릭, 영역 드래그 선택)
- 이동 (드래그)
- 크기 조절 (선택 핸들 드래그)
- 복사/붙여넣기 (Ctrl+C / Ctrl+V)
- 실행 취소/다시 실행 (Ctrl+Z / Ctrl+Y)
- 삭제 (Delete / Backspace)
- 레이어 순서 조절 (앞으로/뒤로 보내기)

### 5. 패널 UI
#### 좌측 패널 — 도형 팔레트
- 도형 버튼 나열
- 클릭 후 캔버스에 드래그하여 배치

#### 우측 패널 — 속성 패널
- 선택된 도형의 속성 표시 및 편집
- 위치 (x, y)
- 크기 (width, height)
- 색상 (fill, stroke)
- 텍스트 입력
- 폰트 크기
- 투명도
- 회전 각도

#### 상단 툴바
- 선택 도구 / 손 도구 (패닝)
- 실행 취소 / 다시 실행
- 저장 (JSON 내보내기)
- 불러오기 (JSON 가져오기)
- PNG 내보내기
- 격자 표시 토글
- 스냅 설정 (격자에 스냅)

### 6. 저장/불러오기
- 레이아웃 데이터를 JSON 형식으로 브라우저 로컬에 저장
- JSON 파일 내보내기 / 가져오기
- PNG 이미지로 내보내기

## 렌더링 방식 검토

| 항목 | Canvas API | SVG |
|------|-----------|-----|
| 성능 (도형 많을 때) | 우수 | 도형 수에 비례하여 저하 |
| 이벤트 처리 | 직접 구현 필요 | 각 요소에 이벤트 직접 부착 가능 |
| 텍스트 렌더링 | 기본적 | 풍부함 |
| PNG 내보내기 | 용이 | 추가 처리 필요 |
| DOM 접근 | 불가 | 가능 |

**결정: SVG 방식 채택**
이유: 도형 수가 많지 않을 것으로 예상되고, 이벤트 처리 및 속성 편집이 간편하며 텍스트 지원이 우수함.

## 메모

- 스냅 기능은 격자 단위로 위치를 맞추는 기능 (토글로 on/off)
- 단위 표시는 픽셀 또는 미터(m) 선택 가능하게 추후 고려
- 모바일 지원은 Phase 1에서는 고려하지 않음 (데스크탑 우선)
